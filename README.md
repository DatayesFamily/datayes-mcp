# Datayes MCP：赋能金融智能体让 AI 拥有专业的金融"大脑"与"手脚"

Datayes MCP，将通联数据沉淀多年的权威金融数据与专业分析能力，封装为标准、安全、 易用的 MCP 服务，让 AI 智能体能够像人类专家一样，自主、可控地获取数据、分析市场、执行策略。

## 为什么选择 Datayes MCP？

**权威数据底座**<br>
背靠通联数据，提供覆盖全市场、多资产类别的权威、高质量金融数据。

**金融场景深度理解**<br>
产品设计源自对投研、量化、风控等核心金融工作流的深刻洞察，工具更专业、更贴合业务。

**企业级安全治理**<br>
从智能网关、权限中间件到全链路日志审计，构建了完整的安全与合规控制体系。

**开放与标准化**<br>
严格遵循 MCP 协议，可轻松对接 Openclaw、Cursor、Trae 等各类主流 AI 智能体与平台，避免生态锁定。

## Datayes MCP 类别介绍

- **基础类 - A 股**：提供 A 股市场数据，涵盖 A 股基础信息、财务及经营数据、行情及交易、股东股本、以及重大事项信息。

- **基础类 - 基金**：提供基金市场数据，涵盖基金基础信息、业绩行情、全品类持仓、以及财务权益信息。

- **基础类 - 宏观行业**：提供宏观行业指标与数据库，涵盖国内及海外主要地区经济统计、货币政策数据、行业级经济指标等。

- **基础类 - 指数**：提供指数市场数据，涵盖指数基础信息、行情与分析信息。

- **基础类 - 公告**：提供沪深京上市公司各类公告信息，及不同时期基金市场公告信息。

- **高阶特色类 - 基金高阶**：提供基金市场高阶信息，涵盖各类风险绩效分析数据。

## Datayes MCP 服务介绍

|服务名称|服务介绍|
|---|---|
|datayes\-stock\-info\-mcp|获取股票证券信息、公司概况、所属行业、员工构成、管理层、实控人等基本信息。|
|datayes\-stock\-financials\-mcp|获取上市公司利润表、资产负债表、现金流量表、杜邦分析等财务指标及经营信息。|
|datayes\-stock\-market\-mcp|获取沪深京股票不同周期的基础行情数据，估值、技术指标、资金流向等多维度行情分析数据以及融资融券、大宗交易、龙虎榜等市场交易行为明细及统计数据。|
|datayes\-stock\-shareholders\-mcp|获取上市公司股东结构、股本变动、股权质押、限售解禁、机构持股及股东大会等多维度的权益数据。|
|datayes\-stock\-events\-mcp|获取上市公司发行上市、诉讼违规、再融资、分红、股权激励、股份回购等重大事件信息。|
|datayes\-fund\-master\-mcp|获取基金基本信息、产品分类、基金经理、发行上市、费率、基金公司、业绩基准配置、重大事件及管理规模等基础与运营信息。|
|datayes\-fund\-performance\-mcp|获取基金净值、行情、复权因子、收益排名及基金经理业绩等表现数据。|
|datayes\-fund\-holding\-mcp|获取基金资产配置、持仓明细、行业与券种分布、投资组合变动以及 QDII 持仓、ETF 申赎运作等相关数据。|
|datayes\-fund\-financials\-mcp|获取基金财务报表、主要财务指标、利润分配、公允价值变动、份额变动及持有人结构等相关数据。|
|datayes\-macro\-mcp|整合国内国际宏观行业指标信息，支持自然语言语义问答，可根据用户提问，智能检索、快速提取中国宏观、海外主要地区、行业经济数据。同时，包含经济日历信息，支持重要经济指标的预测值和预发布日期调用。|
|datayes\-index\-info\-mcp|获取指数的名称、代码、发布机构、加权方式等基本要素息信及指数的成分构成、权重分布等数据。|
|datayes\-index\-market\-mcp|获取指数不同周期的基础行情数据以及估值、业绩表现、资金流向等多维度分析指标。|
|datayes\-stock\-announcements\-mcp|整合沪深京上市公司公告信息，支持自然语言语义问答，可根据用户提问，智能检索、快速提取公告中财报、重大事项、股东大会决议等关键信息。|
|datayes\-fund\-announcements\-mcp|获取基金不同时期的公告信息，包括公告发布时间、公告标题、公告 txt 文本等。|
|datayes\-fund\-analytics\-mcp|获取基金风险收益、回撤指标、股债风格特征、持仓归因、Brinson 归因、行业与择时选股能力等风险绩效分析数据。|
